Connecting to the monitoring hardware

The first time the program is used the IP addresses and Ports of the DCC and RS-Bus monitor hardware should be entered via the Preferences window.

The DCC monitoring server runs on the WIZ-105-SR module on the hardware board described on EasyEda. The IP address and port of that module should be provided via the configuration software belonging to the WIZ serial-Ethernet module. The same address and port should be entered in the preference window of this program.

Although this has not been tested, it should nowadays also be relatively easy to build new monitoring hardware using an ESP Wifi module.

To analyse RS-Bus messages this program should communicate with a Lenz 23151 LAN Interface. Other types of interfaces (such as USB interfaces) are not supported. The IP address and port number of the Lenz 23151 LAN Interface should be obtained using Lenz' configuration program

Saving trace files

The preference window can also be used to set the directory where traces files can be stored. Trace files have as extension ".dcc" and as file name the current date and time.
